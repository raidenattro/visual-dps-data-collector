## 一、代码提交（已入库）

### 1. 上传推测评估服务、API 与 CLI（`32e9106`）

- **目标**：支持将外部/导出的 clip 级推测 JSON（格式同 `localdata/export/rule-baseline-prod-test/*.json`）与本地 `event_review.verified_true` 标真对比，输出召回率、漏报、误报等指标。
- **评估规则**（与 `api/accuracy_service.py` 段级逻辑一致）：
  - **`is_picking=true`** 视为碰撞告警；
  - 货框优先 **`rule_alarm_collisions`**，为空则回退 **`rule_collisions`**；
  - 货框按 **`box_id` 兼容**（如 `85:4017` 与 `Box_4017` 等价，`event_engine.box_identity`）。
  - **漏报（FN）**：标真段内无匹配 `is_picking` 告警；**误报（FP）**：`is_picking=true` 但不在标真段时间+货框范围内。
- **新增**：`api/inference_eval_service.py`、`scripts/data/evaluate_inference_upload.py`。
- **API**：`POST /api/accuracy/evaluate-upload`、`POST /api/accuracy/evaluate-upload-path`。

### 2. 准确率页上传评估 UI（`dd049b9`）

- 评估模式新增 **「上传推测结果评估」**；支持 **单个/多个 JSON 文件** 或 **整个文件夹**（`webkitdirectory`，二者互斥清空）。
- `web/accuracy.js` 以 `FormData` 提交；汇总表与分片明细复用现有准确率页渲染。

### 3. 评估报告落盘（`fe91517`）

### 4. 评估诊断、eval run 落盘与历史查询 API（`8570840`）

- **`api/eval_diagnostics.py`**：完整 FN/FP 列表（无截断）+ `playback_overlay`（`segments` + `alarms` + `source_label` + `counts`）。
- **`api/eval_run_store.py`**：评估结果落盘至 `localdata/eval_runs/{eval_id}/`（`manifest.json`、`summary.json`、`clips/*.json`）。
- **上传评估与本地 timeline 评估**均写入 eval run；本地评估 overlay 来源标签为 `{pose_tier} · timeline 告警`，上传为 `上传推测 · is_picking`。
- **API**（`api/routes/http.py`）：
  - `GET /api/accuracy/eval-runs`
  - `GET /api/accuracy/eval-runs/{eval_id}`
  - `GET /api/accuracy/eval-runs/{eval_id}/clips/{record_id}`
- **回放基础能力**：`navigateToPlaybackRecord` 支持 `seekFrameIdx`、`accuracyOverlay`；`setExternalPlaybackAccuracyOverlay` 注入外部 overlay。

### 5. 准确率页诊断面板与 UI 修复（`e35e94b`）

## 三、诊断 + 回放流程（已实现）

```mermaid
flowchart LR
  A[准确率页评估] --> B[eval_runs 落盘]
  B --> C[分片诊断列表]
  C --> D[跳转回放 seekFrameIdx]
  D --> E[注入 playback_overlay]
  E --> F[事件栏来源统计 + 描边/marker]
```

| 能力 | 说明 |
|------|------|
| 完整 diagnostics | `missed_segments` / `false_alarms` 无截断 |
| eval run 落盘 | `localdata/eval_runs/{eval_id}/` |
| 回放 overlay | 标真段 + 评估源告警，与 evaluate_segments 一致 |
| 来源统计 | 告警/碰撞/标真/漏报段/误报 + `来源：…` 后缀 |
| 历史 run 查询 | API 已就绪；准确率页历史 run 下拉（待做） |

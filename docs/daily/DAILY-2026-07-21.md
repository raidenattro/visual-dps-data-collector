## 二、实现概览

### 1. 核心模块 `spatial_pose/`

| 文件 | 说明 |
|------|------|
| `calibration.py` | 单应标定、`ground_map_bounds()`、volume 运行时 |
| `volume_calibration.py` | 8 角点立体空间、层线、列线反算 |
| `column_grid.py` | 纵深列边界（BL–FL / BR–FR）、`boundaries_y_m`、`column_axis: y` |
| `floor_projection.py` | 踝点 floor_xy、手腕 face_yz、列/层判定 |
| `grid.py` / `preview.py` / `schema.py` | 地面网格、预览、配置 schema |

### 2. 采集与存储

- `config.json` → `spatial.enabled`、`spatial.calibration_dir`
- `collect_core.py`：采集时写 **`floor_foot.parquet`** + **`wrist_face.parquet`** sidecar
- `floor_foot_store.py` / `wrist_face_store.py`：sidecar 读写、`trail_segment_id`
- `pose_store.py`：v2 落盘写入 `wrist_face.parquet`
- `manifest.json` 增加 `spatial` 摘要（`volume_enabled`、`floor_xy_enabled` 等）

### 3. API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/spatial/cameras` | 可标定机位列表 |
| GET/PUT | `/api/spatial/calibration/{slug}` | 读写标定 |
| GET | `/api/spatial/calibration/{slug}/preview-frame` | 推理分辨率预览帧 |
| POST | `/api/spatial/calibration/{slug}/preview` | 预览网格/volume/列线 RMSE |
| GET | `/api/records/{id}/spatial` | 记录 + 运行时标定（含 volume 线框） |
| GET | `/api/records/{id}/floor-foot` | 足部 floor_xy sidecar |
| GET | `/api/records/{id}/wrist-face` | 左右手腕侧面 Y×Z 轨迹 sidecar |

### 4. Web 地面标定页

- 新页签「地面标定」：`web/spatialCalibrate.js`（v18）
- 流程：10 点地面 → volume 8 角点 → **列线吸附 BL–FL(3) / BR–FR(1)** → 左右层线
- 有手标列线时隐藏 homography 横向网格，避免双重绿线
- `preview-frame` 按推理高（480）缩放，与 RTMPose 一致

### 5. 回放 UI

- `web/app/14-playback-floor-map.js`（v15）：
  - **Ground Map**：足部轨迹 + **纵深列分割线**（C1/C2/C3 虚线）
  - **Left/Right Map**：读 `wrist_face.parquet` 画手腕 Y×Z 拖尾
  - 视频 overlay：地面网格 + **列线**（虚线绿）+ 立体空间线框 + 足部/手腕轨迹
  - 加载 `/spatial`、`/floor-foot`、`/wrist-face`；有 volume 时自动勾选「立体空间」
- `web/app/10-render-collision.js`（v89）：修复 overlay 帧号 `fi` 作用域导致骨架消失
- 工具栏：「地面网格」「足部轨迹」「立体空间」

## 三、问题修复

| 问题 | 原因 | 修复 |
|------|------|------|
| Web 标定「未返回视频帧」 | API 返回 dict 而非 base64；路由被 `{slug}` 吞掉 | 修正返回格式；路由提前注册 |
| 标定与采集分辨率不一致 | 原视频 2560×1440 用于 852×480 骨架 | preview-frame 按 infer_height 缩放 |
| 回放地面网格对不齐 | manifest 缺 infer 尺寸 | 从 timeline 读 infer 尺寸 |
| 轨迹跨人/跳变乱连 | 无 track ID | `StickyFootTracker` + `trail_segment_id` |
| 轨迹显示全量历史 | 从第 1 帧画至当前 | 拖尾窗口 `trail_tail_frames` |
| 底部双重绿线 | homography 横线与手标列线重叠 | 有列线时隐藏 homography 横线 |
| 回放无 Map / 无 spatial 请求 | `14-playback-floor-map.js` 语法错误整脚本未加载 | 删除重复代码块；v14+ |
| 勾选立体空间骨架消失 | `drawSkeleton` 中 `fi` 块级作用域 ReferenceError | 提取 `playbackOverlayFi`；v89 |
| Ground Map 列线不显示 | `floorMapToPixel` 用物理像素未用 CSS 坐标；视频 overlay 未画 `column_lines_image` | 对齐 CSS 尺寸；Ground Map 虚线 + 视频列线 overlay；v15 |

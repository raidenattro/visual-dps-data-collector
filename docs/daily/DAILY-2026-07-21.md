## 二、实现概览

### 1. 核心模块 `spatial_pose/`

| 文件 | 说明 |
|------|------|
| `calibration.py` | 单应标定、`ground_map_bounds()` 网格边界 |
| `floor_projection.py` | 踝点选取、floor_xy 投射、EMA 平滑、`StickyFootTracker` |
| `grid.py` / `preview.py` / `schema.py` | 地面网格、预览、配置 schema |

### 2. 采集与存储

- `config.json` → `spatial.enabled`、`spatial.calibration_dir`
- `collect_core.py`：采集时按 `camera_slug` 投射 floor_xy，写入 **`floor_foot.parquet` sidecar**
- `floor_foot_store.py`：sidecar 读写、`trail_segment_id` 分段
- `pose_store.py`：timeline 不再持久化 floor 列（迁移后清理）
- `manifest.json` 增加 `spatial` 摘要与 `files.floor_foot`

### 3. API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/spatial/cameras` | 可标定机位列表 |
| GET/PUT | `/api/spatial/calibration/{slug}` | 读写标定 |
| GET | `/api/spatial/calibration/{slug}/preview-frame` | 推理分辨率预览帧 |
| POST | `/api/spatial/calibration/{slug}/preview` | 预览网格与 RMSE |
| GET | `/api/records/{id}/spatial` | 记录 + 运行时标定 |
| GET | `/api/records/{id}/floor-foot` | 足部 floor_xy sidecar 轨迹 |

### 4. Web 地面标定页

- 新页签「地面标定」：`web/spatialCalibrate.js`
- 10 点交互标点、physical 参数、预览网格、保存至 `localdata/spatial/{slug}.json`
- `preview-frame` 按推理高（默认 480）缩放，与 RTMPose 采集分辨率一致

### 5. 回放 UI

- `web/app/14-playback-floor-map.js`：Ground Map 面板、地面网格 overlay
- **足部轨迹拖尾**（Tail，Handoff 风格）：
  - 仅显示最近 `trail_tail_frames`（默认 90 帧）轨迹，非全量足迹
  - 视频 canvas + Ground Map **共用** `playbackFootTrailPointsForFrame()` 与透明度渐变（旧→淡、新→浓）
  - Ground Map 轨迹受「足部轨迹」开关控制，网格内 clip 裁剪
  - 网格外不显示当前位置 marker / 坐标
  - 播放时随视频延伸，**暂停时轨迹与判定点同步停止**
- 工具栏开关：「地面网格」「足部轨迹」

### 6. CLI / 离线补算

| `scripts/spatial/calibrate_from_video.py` | OpenCV 交互标定 |

| `scripts/spatial/enrich_manifest_floor_xy.py` | 按 export manifest 批量补算 + 回写 export JSON |

文档：`docs/spatial-calibration.md`；模板：`examples/spatial.template.json`

## 三、问题修复

| 问题 | 原因 | 修复 |
|------|------|------|
| Web 标定「未返回视频帧」 | API 返回 dict 而非 base64；`preview-frame` 路由被 `{slug}` 吞掉 | 修正返回格式；路由提前注册 |
| 标定与采集分辨率不一致 | 原视频 2560×1440 标点用于 852×480 骨架 | preview-frame 按 infer_height 缩放；加载旧标定时缩放控制点 |
| 回放地面网格对不齐 | manifest 缺 infer 尺寸；播放路径未画网格 | 从 timeline 读 infer 尺寸；`drawSkeletonPlaybackOnly` 叠加网格 |
| 轨迹跨人/跳变乱连 | 无 track ID，按时间连点 | `StickyFootTracker` + `trail_segment_id` 分段断线（commit `328f5c1`） |
| 轨迹显示全量历史足迹 | 从第 1 帧画至当前帧 | 拖尾窗口 `trail_tail_frames`；Ground Map 与视频同步 |
| 网格外仍计算/显示 floor_xy | 使用较大 `floor_bounds_m` | 改为 `ground_map_bounds()`（grid 2.0×9.6 m）；界外不写 sidecar |

> 实验：基线告警 + **组合3** 段级确认  
> 参考：组合1（disp/fc≤2.5）  

## 组合3 规则

| 条件 | 阈值 |
|------|------|
| `frame_count` | ≥ 4 |
| `duration_sec` | ≥ 0.2 |
| `displacement / frame_count` | ≤ 2.5 |
| `displacement`（总位移） | ≤ 10.0 |

4 项 **同时满足**（AND）；段来自碰撞段提取（`max_gap_frames=1`）。

- 组合3 在 min=5 基线上将误报由 **227** 降至 **56**（-171）。
- 相对组合1：误报 139 → 56（-83）；漏报 23 → 87（+64）。
- 漏报由 **19** 增至 **87**（+68）；召回 88.7% → 48.2%。

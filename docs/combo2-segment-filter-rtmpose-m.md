## 组合2 规则

| 条件 | 阈值 |
|------|------|
| `frame_count` | ≥ 4 |
| `duration_sec` | ≥ 0.2 |
| `displacement / frame_count` | ≤ 3.0 |

三项 **同时满足**（AND）；段来自碰撞段提取（`max_gap_frames=1`）。

## 1. 系统级准确率（标真段 vs 告警）

| 指标 | min=5 基线 | min=5 + 组合2 | 变化 | 组合1 参考 |

### 解读

- 组合2 在 min=5 基线上将误报由 **227** 降至 **144**（-83）。
- 相对组合1：误报 139 → 144（5）；漏报 23 → 22（-1）。
- 漏报由 **19** 增至 **22**（+3）；召回 88.7% → 86.9%。
